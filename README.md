# Cultural Heritage Collages CLI

A CLI tool for finetuning image generation models (LoRA) and performing image inpainting, leveraging Modal for GPU acceleration.

## Setup

1.  Clone the repository.
2.  Ensure you have Python 3.10+ installed.
3.  Install `uv`: `pip install uv`
4.  Create the virtual environment and install dependencies: `uv sync --all-extras`
